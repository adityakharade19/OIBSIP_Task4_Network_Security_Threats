# OIBSIP_Task4_Network_Security_Threats
Research report on common network security threats completed as part of the Oasis Infobyte Cybersecurity Internship.

# Common Network Security Threats

## Objective
To study common network security threats and understand their impact on organizations and individuals.

## Steps Performed
1. Researched common network security threats.
2. Studied DoS attacks, MITM attacks, and spoofing.
3. Analyzed their working mechanisms and impacts.
4. Explored prevention and mitigation techniques.

## Tools Used
- Internet Research
- GitHub

## Outcome
Gained knowledge of common network attacks, their effects, and the best practices used to defend against them.
